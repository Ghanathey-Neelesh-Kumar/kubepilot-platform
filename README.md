# KubePilot Platform

KubePilot is a production-style DevSecOps and Platform Engineering portfolio project built on Kubernetes.

It demonstrates how modern engineering teams can build, deploy, secure, observe, and troubleshoot applications using cloud-native practices.

## What This Project Includes

- Kubernetes application deployment
- Docker-based local development
- GitHub Actions CI/CD
- GitOps deployment with Argo CD
- Container image scanning with Trivy
- Policy enforcement with Kyverno
- Monitoring with Prometheus
- Dashboards with Grafana
- Logging with Loki and Promtail
- Alerting with Alertmanager
- Kubernetes troubleshooting labs
- KubePilot Doctor troubleshooting assistant
